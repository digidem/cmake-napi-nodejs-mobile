# cmake-napi

```
npm i cmake-napi
```

```cmake
find_package(napi REQUIRED PATHS node_modules/cmake-napi)
```

## API

#### `find_node(<result>)`

#### `download_node_headers(<result> [VERSION <version>] [DESTINATION <path>])`

#### `napi_module_target(<directory> <result>)`

#### `add_napi_module(<result>)`
